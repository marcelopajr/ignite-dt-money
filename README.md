## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/marcelopajr/ignite-dt-money.git

- Enter directory:
$ cd ignite-dt-money

- Install dependencies:
$ npm install

- Launch the app:
$ npm run dev

- Launch the server:
$ npm run dev:server
```

Have fun!
